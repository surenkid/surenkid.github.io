## 关于自己

- **坐标南京**，一名踏实的IT从业者，生于斯长于斯。
- **技术多面手**，入行十六年，从修电脑、配网络到开发软件、维护系统，属于软硬件都能上手的“多面手”。
- **深耕Web领域**，拥有十多年的实战经验，擅长各类网站建设和微信小程序开发，这是我吃饭的手艺，也是我最擅长的领域。
- **终身学习者**，喜欢钻研新鲜事物，更热衷于搞懂背后的原理，致力于把复杂的技术转化为简单好用的工具。
- **务实派**，不空谈理论，只想用代码解决实际问题，并在敲击键盘中寻找生活的平衡。

| 项目     | 名称                | 备注         |
| -------- | -------------       | ------------ |
| 服务器   | Github Pages        | 静态托管     |
| 域名     | hi.ktsee.uum.cc     |              |

博客的副标题选用了乔布斯说的：**Stay hungry,Stay foolish**，我很喜欢这句话，也希望能勉励自己不断学习。
